- **Language**: JavaScript ES6+

## 🚀 Installation Methods

### 1. Manual Installation
```bash
git clone https://github.com/sobri3195/whatsapp-bot-qr.git
cd whatsapp-bot-qr
npm install
npm start
```

### 2. Quick Setup Script
```bash
./setup.sh
npm start
```

### 3. Docker (Optional)
```bash
docker-compose up -d
```
